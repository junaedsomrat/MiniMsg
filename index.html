Disclaimer

Please read before using MiniMsg.

1. Serverless Nature: MiniMsg operates entirely in your browser. We do not have a backend server or database. All text and data you input are encrypted and stored directly inside the generated URL.

2. User Responsibility: Because the data exists only in the link, we have no control over the content shared. The user generating the link is solely responsible for the content distributed.

3. Lost Links & Passwords: We cannot recover lost links or forgotten passwords. If you lose the URL or the decryption password, the text is permanently unrecoverable.

4. Browser Limits: Extremely long texts may exceed the maximum URL length permitted by some browsers. Please keep your sharing concise.
